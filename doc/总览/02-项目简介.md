# 项目简介

## 项目概述

AI-BE-PY 是 AI-BE 的 Python 重构版本，基于 LangGraph 构建的航运领域智能对话微服务。

## 项目背景

AI-BE（Java 版）是面向航运等业务场景的 AI 后端服务，提供：
- 智能对话
- 意图识别与场景路由
- 工具调用（MCP、运单、天气等）
- RAG 知识增强
- 用户记忆系统

AI-BE-PY 将使用 Python + LangGraph 重构核心链路，以支持：
- 更灵活的 Agent 编排
- 更快速的业务迭代
- 与现有 AI 生态（LangChain、LangGraph、MCP）无缝集成

## 主要功能

| 功能 | 说明 |
|------|------|
| 流式对话 | 统一对话入口，支持 SSE 流式响应 |
| 意图识别 | 规则网关 + 大模型分类 |
| 场景编排 | LangGraph 状态机驱动 |
| 工具调用 | MCP 协议 + 业务工具 |
| 知识增强 | RAG 向量检索 |
| 记忆系统 | 用户画像 + 会话记忆 |

## 业务场景

| 场景 | 说明 |
|------|------|
| DOC_QA | 文档问答 |
| FIND_SHIP | 找船 |
| SAVE_ORDER | 运单录入 |
| QUERY_ORDER | 历史订单查询 |
| QUERY_SHIP | 查船 |
| QUERY_FREIGHT | 运价查询 |
| QUERY_WEATHER | 天气查询 |

| 类别 | 技术 |
|------|------|
| 语言 | Python 3.11+ |
| 框架 | FastAPI + LangGraph |
| AI | DeepSeek / OpenAI |
| 向量 | Milvus |
| 缓存 | Redis |
| 数据库 | MySQL |
| 协议 | MCP |

## 与 SpringAI 版本对比

| 维度 | SpringAI 版本 | Python 版本 |
|------|---------------|-------------|
| 编排方式 | FlowEngineProcessor | LangGraph |
| 状态管理 | SceneContext | Graph State |
| 工具调用 | IToolService | Tools + MCP |
| 扩展性 | 中 | 高（Python 生态） |
| 学习成本 | 中 | 低 |
