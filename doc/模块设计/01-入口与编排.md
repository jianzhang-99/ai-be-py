# 01-入口与编排

## 模块职责

接收用户请求，入口编排，协调整个对话流程。

## 对应原项目

- `FlowEngineProcessor.java`

## 主流程

```
用户请求 → API Layer → Services → Graph Workflow → Response
```

## Python 实现

### 入口文件

`backend/main.py`

```python
from fastapi import FastAPI
from fastapi.responses import StreamingResponse
from backend.api.routers import chat, health

app = FastAPI(title="AI-BE-PY", version="0.1.0")

app.include_router(chat.router)
app.include_router(health.router)

if __name__ == "__main__":
    import uvicorn
    uvicorn.run("backend.main:app", host="0.0.0.0", port=8000, reload=True)
```

### API 层

`backend/api/routers/chat.py`

```python
from fastapi import APIRouter, Depends
from backend.api.deps import get_chat_service
from backend.api.schemas import ChatRequest, ChatResponse

router = APIRouter(prefix="/api", tags=["chat"])

@router.post("/chat/stream")
async def chat_stream(
    request: ChatRequest,
    service = Depends(get_chat_service)
):
    """SSE 流式对话接口"""
    return StreamingResponse(
        service.chat_stream(request),
        media_type="text/event-stream"
    )
```

### 工作流入口

`backend/graph/workflows/chat_workflow.py`

```python
from langgraph.graph import StateGraph, END
from backend.graph.state.agent_state import AgentState

def create_chat_workflow():
    """创建对话工作流"""
    workflow = StateGraph(AgentState)

    # 添加节点
    workflow.add_node("intent", intent_node)
    workflow.add_node("route", routing_node)
    workflow.add_node("tool", tool_node)
    workflow.add_node("rag", rag_node)
    workflow.add_node("memory", memory_node)
    workflow.add_node("response", response_node)

    # 设置边
    workflow.set_entry_point("intent")
    workflow.add_edge("intent", "route")

    # 条件边：路由到不同节点
    workflow.add_conditional_edges(
        "route",
        route_decision,
        {
            "tool": "tool",
            "rag": "rag",
            "response": "response"
        }
    )

    workflow.add_edge("tool", "response")
    workflow.add_edge("rag", "response")
    workflow.add_edge("response", "memory")
    workflow.add_edge("memory", END)

    return workflow.compile()
```

## 设计要点

1. **流式响应**: 使用 FastAPI StreamingResponse + SSE
2. **状态传递**: 通过 LangGraph State 管理对话状态
3. **节点解耦**: 每个节点独立，可单独测试
