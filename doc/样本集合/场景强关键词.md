# 场景强关键词

## 匹配规则

- 对用户输入做**子串匹配**（`contains`）。
- 按下方 **扫描顺序** 依次检查，**先命中先返回**。
- 未列入扫描顺序的场景不参与本次强关键词命中。

| 顺序 | Code                | 场景说明           | 强关键词                                                     |
| :--: | :------------------ | :----------------- | :----------------------------------------------------------- |
|  1   | `DOC_QA`            | 文档问答、操作指引 | 怎么用、怎么操作、如何使用、怎么做、步骤是什么、怎么弄、运吨吨是什么、介绍一下自己、详细介绍一下运吨吨 |
|  2   | `FIND_SHIP`         | 找船               | 找船、有没有船、找一下船、匹配船、匹配运力、附近的船、航线找船 |
|  3   | `SAVE_ORDER`        | 运单录入           | 发布运单、发运单、我要发货、我要下单、录入运单、创建运单、保存运单、发布订单、发订单、录入订单 |
|  4   | `QUERY_ORDER`       | 历史订单查询       | 查订单、我的订单、订单查询、历史订单、运单查询、我的运单、历史运单 |
|  5   | `QUERY_SHIP`        | 查船               | 查询船舶、船在哪、船到哪、船舶位置                           |
|  6   | `QUERY_FREIGHT`     | 运价查询           | 运价多少、价格多少、运费多少、费用多少、给我报价             |
|  7   | `QUERY_WEATHER`     | 天气查询           | 天气、下雨吗、气温多少                                       |
|  8   | `QUERY_WATER_LEVEL` | 水位查询           | 水位、水深                                                   |
|  9   | `DISPATCH_MONITOR`  | 在途监控           | 在途监控、运输进度、物流进度、运输状态                       |
|  10  | `IMAGE_OCR`         | 图像识别           | 识别图片、图片识别、识别一下、帮我识别、拍照识别、提取文字   |
|  11  | `FEEDBACK`          | 反馈               | 反馈、投诉、建议、有问题、报错、有 bug、不好用               |
|  12  | `TALK`              | 闲聊，对话         | 你好、在吗、哈哈、聊聊天                                     |

## 代码中的扫描顺序常量

`STRONG_KEYWORD_ORDER`：

`DOC_QA` → `FIND_SHIP` → `SAVE_ORDER` → `QUERY_ORDER` → `QUERY_SHIP` → `QUERY_FREIGHT` → `QUERY_WEATHER` → `QUERY_WATER_LEVEL` → `DISPATCH_MONITOR` → `IMAGE_OCR` → `FEEDBACK` → `TALK`
