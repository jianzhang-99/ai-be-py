## 样本一：上下文意图切换失败

**错误样本** 专有名词歧义 —— "俞垛79"是船舶名，"俞垛"是地名（江苏泰州靖江市俞垛镇）

### 第一轮

| 角色 | 内容 |
|------|------|
| 用户 | 查船 俞垛79 |
| AI | 已为您查询到船舶：俞垛79，船舶当前位置已在地图上标注展示 |
| AI 地图卡片 | 船名：俞垛79, MMSI: 413774544，位于扬州万福闸风景区附近 |

### 第二轮

| 角色 | 内容 |
|------|------|
| 用户 | 俞垛在哪里 |
| AI | 未查询到完全一致的船舶信息，为您推荐相似船舶：俞垛777 |
| AI 地图卡片 | 船名：俞垛777, MMSI: 413861861，位于江海湿地公园附近的长江航道上 |

**问题分析：** 用户问"俞垛在哪里"是想问"俞垛79在哪里"，但系统误解为查询"俞垛镇在哪里"，找不到对应船舶后推荐了"俞垛777"。意图识别模块未正确关联上下文中的船舶名。

---


**正确样本** 上下文冲突 —— 上文刚查询船舶位置，下文问"在哪里"被理解为地名查询

### 第一轮

| 角色 | 内容 |
|------|------|
| 用户 | 俞垛79在哪里 |
| AI | 已为您查询到船舶：俞垛79，船舶当前位置已在地图上标注展示 |
| AI 地图卡片 | 船名：俞垛79, MMSI: 413774544，请查看地图位置，显示长江沿岸，中心标注为"俞垛79" |

### 第二轮

| 角色 | 内容 |
|------|------|
| 用户 | 俞垛在哪里 |
| AI | 俞垛是位于中国江苏省泰州市靖江市的一个镇，地处长江北岸，是重要的内河航运节点之一 |
